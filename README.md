# Detector de Uso de Casco en Motociclistas
Proyecto desarrollado para el curso de Inteligencia Artificial.

Descripción
Sistema de visión artificial basado en YOLOv8 capaz de detectar motociclistas con casco y sin casco a partir de imágenes.

Objetivo
Identificar automáticamente el uso correcto del casco de seguridad en motociclistas utilizando técnicas de Deep Learning y detección de objetos.

Dataset
Bikes Helmets Dataset
Clases:
With Helmet
Without Helmet
Tecnologías utilizadas
Python
YOLOv8
PyTorch
Streamlit
OpenCV

Resultados
| Métrica   | Valor |
| --------- | ----- |
| Precision | 73.3% |
| Recall    | 80.1% |
| mAP50     | 77.3% |
| mAP50-95  | 47.0% |


Ejecución
Instalar dependencias:
pip install -r requirements.txt
Ejecutar aplicación 
streamlit run app.py
